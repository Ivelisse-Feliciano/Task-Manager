# Business Productivity Dashboard

## Project Overview

The Business Productivity Dashboard is a web application built with Next.js and React that helps users organize, prioritize, and track daily tasks. Users can categorize tasks by Work, School, Church, or Personal, assign due dates, monitor completion status, and track overall productivity through a clean, responsive dashboard. The application stores task data locally, allowing users to maintain their task list between browser sessions.

---

## Technologies Used

- Next.js (App Router)
- React
- JavaScript (ES6+)
- Tailwind CSS
- Local Storage API
- Git & GitHub
- npm

---

## Running the Project

### Clone the repository

```bash
git clone https://github.com/Ivelisse-Feliciano/task-manager.git
```

### Navigate to the project

```bash
cd task-manager
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open your browser and navigate to:

```text
http://localhost:3000
```
